# Paradise Nursery

**Paradise Nursery** est une application faite avec **React** et qui permet aux utilisateur de pouvoir choisir des variétés de plantes d'intérieur qu'ils désirent acheter.

L'application contient :

- **Une page d'atterissage**
- **Une page de listage de produit**
- **Une page du pagnier d'achat**

Le dépot **GitHub** du projet "_e-plantShopping_" se trouve ici [GitHub](https://github.com/JOHN-gif1224/e-plantShopping#)

## Panier — calcul des totaux (`CartItem.jsx`)

Le composant panier implémente et affiche :

- **`calculateTotalCost(item)`** — coût par ligne (prix unitaire × quantité)
- **`calculateTotalAmount(items)`** — montant total du panier
